== Description ==

WooCommerce Wholesale Discount allow you to set wholesale discount on Woocommerce cart items based on its quantity. This plugin is compatible with Woocommerce 2.7.0 and greater versions. 


= New Feature =
Discount can be set against both Product and Category, plugin will first give preference to product wise discount then category. 

= Get free support within 24 hours = 

== Features == 
 
 * Plugin allow option to set discount in percentage and flat to Products wise and Category wise.
 * Plugin allow option to set data range for discount rule.
 * Plugin allow option to disable wholesale discount.
 * Plugin allow option to set discount calculate on Combined product variations.
 * Plugin allow option to add and remove wholesale discount if default Woocommerce coupon code is applied. 
 * Wholesale discount is visible at Cart, Checkout, Order Detail page and order email.

== Installation == 

1. Install the plugin either by uploading it via FTP or by using the "Add Plugin" function of WordPress. 
2. Activate the plugin at the plugin administration page.

After Activation of plugin follow below instruction to made this work:

1. Go to Woocommerce Settings, and then under Wholesale discount tab you can set plugin global settings.
2. Go to Product list in wordpress admin, open a product in edit mode for which you want to set discount, at Product Data section there will be **Wholesale Discount**  tab where you can enable wholesale discount set discount duration, unlimited discount slabs.
3. For Category based discount, you will see a Wholesale Discount tab under Woocommerce menu, where you can set unlimited discount rules against categories.
